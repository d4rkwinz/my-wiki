# Caching

Fast intermediate stores that absorb read load before it reaches origin systems. Spans in-memory caches in front of databases and geographically distributed edges in front of static content.

Parent: [[System Design]]

## Concepts

- [[Cache]] — fast in-memory store for frequently accessed data, sitting between web and data tiers.
- [[CDN]] — geographically distributed cache for static assets (also indexed under [[Networking]]).
